The Project:

If I’m not snapped up by a programming company with morals soon, Raytheon might get me. It’s a camera on a remote controlled swivel. What more do you want from me? This project is similar to my rubber band turret because it uses some of that project’s old hardware. However, my tool-chain has transitioned to socket programming in Rust. This project runs a Rust multi-threaded (ooh, multi-threading *pause for applause) web server to manage a controller client on my laptop as well as a camera feed input from a Raspberry Pi. Don’t get too excited, it’s not well written (I keep writing ”refactor” everywhere though so at least I KNOW it’s not well written).

I have big plans for this project: big water/gel gun plans. I already have a ”fire” flag bit built into the code and sent to the turret. Once I up the speed and improve the camera latency on this bad boy… it will be a cooler project. It also looks kinda cute. The camera looks like a lil eye!

-July 2025

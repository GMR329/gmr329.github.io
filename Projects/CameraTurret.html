Project Update:

I’m now a weapons manufacturer! This whole thing is hacked together and barely functional, but doesn’t that make it way more fun? The code is largely unchanged, though I had to adapt it for certain electrical issues. I originally had purchased an electrical gel blaster and attempted to splice into the switch mechanism and control it with a BJT. I messed up, fried the selective fire board in the gun and had to control the firing motor directly. This motor needed over an amp, and the BJT couldn’t provide it, so I switched to a MOSFET... only to realize the voltage that motor needed was too high for the servo motors. But me, I’m a clever one, and I whipped up a voltage divider circuit. And then I remembered I’m actually not clever and that circuit isn’t made to carry the current I require to the motor. My ultimate solution had a MOSFET used as a switch for the shooting motor and a DC-DC Buck converter to drop the voltage for the servo motors. The control systems code had to change to accommodate direct control of the motor, and everything else remained as it was in the previous, camera only, iteration.

The pew pew also appears rather weak in the video… I probably threw out some important plastic piece and it’s possible the gel balls lost a bit of volume. But it’s still pretty fun to shoot.

-August 2025

The Project:

If I’m not snapped up by a programming company with morals soon, Raytheon might get me. It’s a camera on a remote controlled swivel. What more do you want from me? This project is similar to my rubber band turret because it uses some of that project’s old hardware. However, my tool-chain has transitioned to socket programming in Rust. This project runs a Rust multi-threaded (ooh, multi-threading *pause for applause) web server to manage a controller client on my laptop as well as a camera feed input from a Raspberry Pi. Don’t get too excited, it’s not well written (I keep writing ”refactor” everywhere though so at least I KNOW it’s not well written).

I have big plans for this project: big water/gel gun plans. I already have a ”fire” flag bit built into the code and sent to the turret. Once I up the speed and improve the camera latency on this bad boy… it will be a cooler project. It also looks kinda cute. The camera looks like a lil eye!

-July 2025
